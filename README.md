# AutomotiveClaw

Automotive dealership management for [ERPClaw](https://github.com/avansaber/erpclaw). Vehicle inventory, deal structuring, F&I products, service department, parts management, and regulatory compliance. 70 actions across 8 domains.

## Features

- **Vehicle Inventory** -- new/used/CPO tracking, VIN lookup, trade-in appraisals, aging reports
- **Deal Management** -- retail/lease/wholesale/fleet deals, buyer orders, finalization and unwind
- **F&I Products** -- warranty, GAP, maintenance plans with penetration and income reporting
- **Service Department** -- repair orders, service lines, warranty claims, efficiency reports
- **Parts** -- parts catalog, ordering, receiving, velocity and valuation reports
- **Compliance** -- OFAC screening, buyers guide, odometer statements, compliance checks
- **Customers** -- customer profiles, vehicle history, service history
- **Reports** -- gross profit, salesperson performance, inventory aging, F&I penetration

## Installation

Requires [ERPClaw](https://github.com/avansaber/erpclaw) core.

```
install-module automotiveclaw
```

## Quick Start

```
"I'm opening a car dealership -- set me up"
```

The bot will install AutomotiveClaw and guide you through adding vehicles, structuring deals, and managing your service department.
